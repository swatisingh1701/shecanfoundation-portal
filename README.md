# She Can Foundation Portal

A responsive React-based web application developed for the She Can Foundation Full Stack Development Internship Task.

The portal enables users to submit contact requests through an interactive form and provides an admin dashboard to manage and monitor submissions efficiently.

---

## Features

### User Side
- Submit contact requests
- Name, Email, Subject, and Message fields
- Client-side form validation
- Success notification after submission
- Responsive user interface
- Light/Dark mode support

### Admin Dashboard
- View all form submissions
- Search submissions by name or email
- Delete submissions
- Submission count overview

### Data Management
- Local Storage integration
- Persistent data storage
- Automatic timestamp generation

## Installation & Setup

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd myapp
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
